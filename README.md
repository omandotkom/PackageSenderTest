# Package Sender Test by omandotkom

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

Package Sender Test is a very simple java package sender. The package sent via network (TCP/IP) at the same or different network.
  - Simple usage
  - One single app with two modes

### Usage
- Note : Package Sender Test requires java installed on the computer.

To run as server

```sh
$ javac Main.java
$ java Main server
```

To run as Client

```sh
$ javac Main.java
$ java Main client
```
then input server's ip address.
